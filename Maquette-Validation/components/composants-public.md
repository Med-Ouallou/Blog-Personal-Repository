# Liste des Composants HTML (MVP Public)

Cette liste définit les briques unitaires pour l'interface Publique, organisées par sous-dossiers.

## 1. Structure & Layouts (`layout/`)
*   **`layout-base.html`** (Existant)
    *   Squelette HTML5, Head, Scripts, Variables CSS.
*   **`layout-simple.html`** (Existant)
    *   Version épurée pour Login/Register.

## 2. Navigation (`navigation/`)
*   **`navbar.html`** (Existant)
    *   Menu principal et Gestion état connecté/déconnecté.
*   **`footer.html`** (Existant)
    *   Liens et Copyright.

## 3. Contenus & Blocs (`contenu-cards/`)
*   **`hero.html`** (Existant)
    *   Section d'introduction page d'accueil (Titre, CTA, Image).
*   **`card-article.html`** (Existant)
    *   Carte article standard (Image, Badge, Tags, Méta).
*   **`article-detail.html`** (Existant)
    *   Corps détaillé de l'article avec typographie Prose.
*   **`statistics.html`** (Existant)
    *   Section des chiffres clés (Tutoriels, Étudiants, etc.).
